# Interactive Seminar Hall Simulation (Blender + Unity)

## Project Overview

This project is an interactive 3D seminar hall simulation created using Blender and Unity.

The seminar hall was first modeled in Blender and then imported into Unity, where multiple interactions were implemented to improve user experience and demonstrate basic game development concepts.

## Features Implemented

### 1. Multiple Camera Views

* Entrance View
* Stage View
* Audience View

Users can switch between different perspectives using Unity UI buttons.

### 2. Room Light Toggle

* A room light can be switched ON and OFF using a button.

### 3. Seminar Hall Information Display

* Hall information is displayed using Unity UI.

## Technologies Used

* Blender (3D Modeling)
* Unity 6 (Scene Creation and Interactions)
* C# (Interaction Scripting)
* TextMesh Pro (UI Text)
